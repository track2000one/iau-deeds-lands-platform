## 📋 القائمة الجانبية

### كيف تفتح القائمة؟

**الخطوة 1**: ابحث عن أيقونة الخطوط الثلاثة (☰) في أعلى الشاشة

**الخطوة 2**: اضغط عليها

**النتيجة**: القائمة تنزلق من الجانب! 🎉

```
┌──────────────────────┐
│ ☰ التطبيق      [⚙] │  ← اضغط هنا
└──────────────────────┘
        ⬇️
┌──────────────────────┐
│ [القائمة المنزلقة]  │
│  🏠 الرئيسية        │
│  📄 الصكوك          │
│  📊 التقارير        │
│  ...                │
└──────────────────────┘
```

### كيف تغلق القائمة؟

**طريقة 1**: اضغط على أي عنصر في القائمة
**طريقة 2**: اضغط على الشاشة السوداء خلف القائمة
**طريقة 3**: اضغط على زر X داخل القائمة

## 📊 صفحة التقارير

### التبويبات

على الموبايل، التبويبات موزعة في شبكة 2×2:

```
┌───────────┬───────────┐
│ تفاصيل   │  ملخص     │
├───────────┼───────────┤
│ إحصائي   │  رسوم     │
└───────────┴───────────┘
```

**كيف تستخدمها؟**
1. اضغط على التبويبة المطلوبة
2. المحتوى يتغير تلقائياً
3. التبويبة النشطة بلون مميز

---

### اختيار الأعمدة

**الخطوات**:

```
1️⃣ افتح قسم "اختيار الأعمدة"
        ⬇️
2️⃣ ضع علامة ✓ على الأعمدة المطلوبة
        ⬇️
3️⃣ استخدم "تحديد الكل" للاختيار السريع
        ⬇️
4️⃣ الأعمدة المختارة فقط ستظهر في التقرير
```

**مثال**:
```
☑️ رقم القطعة
☑️ المساحة
☐ الإحداثيات  ← غير محدد
☑️ المدينة
```

---

### أزرار التصدير

على الموبايل، الأزرار تظهر واحداً تحت الآخر:

```
┌──────────────────┐
│   📊 Excel       │  ← تصدير إلى Excel
├──────────────────┤
│   📄 PDF         │  ← تصدير إلى PDF
├──────────────────┤
│   👁️ طباعة      │  ← معاينة وطباعة
└──────────────────┘
```

**كيف تستخدمها؟**
- اضغط على الزر المطلوب
- انتظر قليلاً
- سيتم تحميل الملف تلقائياً!

## 🎨 نصائح وحيل

### 💡 نصيحة 1: التنقل السريع
```
☰ → اختر الصفحة → انتهى!
لا حاجة لإغلاق القائمة يدوياً
```

### 💡 نصيحة 2: الطباعة
```
لأفضل نتيجة:
1. استخدم "معاينة وطباعة"
2. تحقق من المعاينة أولاً
3. اختر "حفظ كPDF" بدلاً من الطباعة المباشرة
```

### 💡 نصيحة 3: التقارير
```
التقرير المفصل: للتفاصيل الكاملة
التقرير الملخص: لنظرة سريعة
الرسوم البيانية: للعروض التقديمية
```

### 💡 نصيحة 4: توفير البيانات
```
اختر الأعمدة الضرورية فقط عند التصدير
هذا يوفر:
- حجم الملف ⬇️
- وقت التحميل ⏱️
- استهلاك البيانات 📶
```

## 🔧 حل المشاكل

### المشكلة: القائمة لا تفتح
**الحل**:
1. تحقق من الاتصال بالإنترنت
2. حدّث الصفحة (↻)
3. امسح الكاش
4. أعد تسجيل الدخول

### المشكلة: الجدول لا يظهر
**الحل**:
1. تأكد من وجود بيانات
2. مرر للأسفل
3. حاول التمرير أفقياً

### المشكلة: الطباعة لا تعمل
**الحل**:
1. تأكد من تفعيل النوافذ المنبثقة
2. جرب حفظ PDF بدلاً من الطباعة
3. استخدم متصفح مختلف

### المشكلة: الصفحة بطيئة
**الحل**:
1. أغلق التطبيقات الأخرى
2. امسح ذاكرة التخزين المؤقت
3. تحقق من سرعة الإنترنت
4. حدّث المتصفح
