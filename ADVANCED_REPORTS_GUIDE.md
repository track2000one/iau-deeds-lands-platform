# 📊 دليل التقارير الاحترافية المتقدمة

## 🎯 نظرة عامة

تم تطوير نظام التقارير ليكون شاملاً واحترافياً ومتطوراً يدعم جميع أنواع التقارير والتحليلات المطلوبة لإدارة الأراضي والممتلكات.

---

## ✨ الميزات الرئيسية

### 1. 📋 أنواع التقارير المتعددة

#### 🔹 تقرير مفصّل (Detailed Report)
**الوصف**: عرض كامل لجميع البيانات مع إمكانية اختيار الأعمدة

**المميزات**:
- ✅ عرض جميع السجلات في جدول منظم
- ✅ اختيار الأعمدة المراد عرضها
- ✅ تحديد/إلغاء تحديد جميع الأعمدة بنقرة واحدة
- ✅ تصدير إلى Excel مع الأعمدة المختارة فقط
- ✅ تصدير إلى PDF
- ✅ طباعة مباشرة مع معاينة

**الاستخدام**:
```
1. اضغط على السهم لتوسيع القسم
2. اختر تبويب "مفصّل"
3. حدد الأعمدة المطلوبة من قسم "اختيار الأعمدة"
4. اضغط على زر التصدير/الطباعة المطلوب
```

---

#### 🔹 تقرير ملخص (Summary Report)
**الوصف**: عرض إحصائيات سريعة ومختصرة

**المميزات**:
- 📊 إجمالي عدد السجلات
- 📐 إجمالي المساحة (للأراضي والمباني)
- 💰 إجمالي الإيجارات (للعقود)
- 🎨 عرض بصري جذاب مع أيقونات

**الاستخدام**:
```
1. اختر تبويب "ملخص"
2. شاهد الإحصائيات الفورية
3. يمكن طباعة الملخص مباشرة
```

**مثال**:
```
┌──────────────────────┐
│  إجمالي السجلات     │
│        125           │
└──────────────────────┘

┌──────────────────────┐
│  إجمالي المساحة     │
│    45,250 م²        │
└──────────────────────┘
```

---

#### 🔹 تقرير إحصائي (Statistical Report)
**الوصف**: تحليل وتوزيع البيانات حسب معايير مختلفة

**المميزات**:
- 🏙️ **التوزيع الجغرافي**: عدد الأراضي/المباني حسب المدينة
- 🏗️ **التوزيع حسب الاستخدام**: أنواع الاستخدام المختلفة
- 👥 **التوزيع حسب الجهات**: المستفيدون والمستأجرون
- 📈 **إحصائيات متقدمة**: معدلات، نسب، مقارنات

**الاستخدام**:
```
1. اختر تبويب "إحصائي"
2. اطلع على التوزيعات المختلفة
3. راجع التحليلات والنسب
```

**أمثلة على التحليلات**:

##### توزيع الأراضي المخصصة حسب المدينة:
| المدينة | العدد |
|---------|-------|
| الدمام   | 45    |
| الخبر   | 32    |
| القطيف  | 28    |

##### توزيع حسب نوع الاستخدام:
| نوع الاستخدام | العدد | النسبة |
|---------------|-------|--------|
| سكني          | 60    | 48%    |
| تجاري        | 30    | 24%    |
| صناعي        | 25    | 20%    |
| خدمات        | 10    | 8%     |

---

#### 🔹 تقرير رسوم بيانية (Graphical Report)
**الوصف**: تمثيل مرئي للبيانات باستخدام الرسوم البيانية

**أنواع الرسوم**:

1. **📊 المخططات الشريطية (Bar Charts)**
   - لعرض التوزيع حسب المدن
   - مقارنة الكميات
   - سهلة القراءة والفهم

2. **🥧 المخططات الدائرية (Pie Charts)**
   - لعرض النسب المئوية
   - توزيع أنواع الاستخدام
   - تمثيل الحصص

**المميزات**:
- 🎨 ألوان متدرجة ومميزة
- 📱 متجاوب مع جميع الشاشات
- 🖱️ تفاعلي مع Tooltips
- 📤 قابل للطباعة والتصدير

**الاستخدام**:
```
1. اختر تبويب "رسوم بيانية"
2. اطلع على المخططات المختلفة
3. مرر الفأرة على الأقسام لرؤية التفاصيل
```

---

### 2. ⚙️ اختيار الأعمدة (Column Selection)

**الوصف**: تحكم كامل في الأعمدة المعروضة والمطبوعة

**الميزات**:
- ✅ اختيار أي عمود أو إلغاء اختياره
- ✅ زر "تحديد الكل" لاختيار جميع الأعمدة
- ✅ زر "إلغاء الكل" لإلغاء جميع الاختيارات
- ✅ عداد يظهر عدد الأعمدة المختارة
- ✅ التصدير يشمل الأعمدة المختارة فقط

**الأعمدة المتاحة لكل نوع**:

#### الأراضي المخصصة:
- رقم القطعة ✓
- رقم المخطط ✓
- المساحة (م²) ✓
- نوع الاستخدام ✓
- المدينة ✓
- الحي ✓
- المنطقة ✓
- الإحداثيات

#### الأراضي المسلمة:
- الجهة المستلمة ✓
- تاريخ الاستلام ✓
- رقم القطعة ✓
- رقم المخطط ✓
- المساحة (م²) ✓
- الموقع ✓
- الإحداثيات

#### الأراضي المؤجرة:
- المستأجر/المالك ✓
- رقم العقد ✓
- التاريخ والمدة ✓
- رقم القطعة ✓
- المساحة (م²) ✓
- قيمة الإيجار ✓

- ✅ الأعمدة المختارة فقط

- ✅ جداول منظمة بألوان

- ✅ حجم A4 أفقي للعرض الأمثل

**الاستخدام**: اضغط على زر PDF 📄

**اسم الملف**: `Report_Name_YYYY-MM-DD.pdf`

- ✅ تذييل بتوقيع وختم
- ✅ حوار الطباعة يفتح تلقائياً

## 📚 أمثلة الاستخدام

### مثال 1: طباعة تقرير مخصص
```
الهدف: طباعة تقرير الأراضي المخصصة مع أعمدة محددة فقط

الخطوات:
1. اذهب إلى صفحة التقارير
2. اضغط على السهم بجانب "الأراضي المخصصة"
3. تأكد من اختيار تبويب "مفصّل"
4. في قسم "اختيار الأعمدة":
   - حدد: رقم القطعة ✓
   - حدد: المساحة ✓
   - حدد: المدينة ✓
   - حدد: الحي ✓
   - ألغِ باقي الأعمدة
5. اضغط على "معاينة وطباعة"
6. راجع المعاينة
7. اضغط طباعة من حوار المتصفح

النتيجة: تقرير مطبوع بالأعمدة المحددة فقط
```

---

### مثال 2: تصدير بيانات للتحليل
```
الهدف: تصدير بيانات الإيجارات إلى Excel للتحليل المالي

الخطوات:
1. افتح قسم "الأراضي المؤجرة"
2. اختر تبويب "مفصّل"
3. حدد جميع الأعمدة (اضغط "تحديد الكل")
4. اضغط زر Excel 📊
5. احفظ الملف
6. افتح في Excel لإجراء التحليلات

النتيجة: ملف Excel كامل جاهز للتحليل
```

---

### مثال 3: عرض إحصائيات سريعة
```
الهدف: معرفة توزيع الأراضي المخصصة حسب المدن

الخطوات:
1. افتح قسم "الأراضي المخصصة"
2. اختر تبويب "إحصائي"
3. راجع قسم "التوزيع حسب المدينة"
4. اضغط تبويب "رسوم بيانية" لرؤية المخطط الشريطي

النتيجة: فهم واضح للتوزيع الجغرافي
```

---

### مثال 4: تقرير شامل مع رسوم بيانية
```
الهدف: إنشاء تقرير تقديمي مع إحصائيات ورسوم

الخطوات:
1. افتح القسم المطلوب
2. اختر تبويب "ملخص" - اطبع الإحصائيات
3. اختر تبويب "رسوم بيانية" - التقط صورة للمخططات
4. اختر تبويب "مفصّل" - اطبع الجدول الكامل
5. اجمع كل الأجزاء في عرض تقديمي

النتيجة: تقرير تقديمي احترافي متكامل
```

## 🎯 أفضل الممارسات

### ✅ للحصول على أفضل النتائج:

1. **قبل الطباعة**:
   - راجع الأعمدة المختارة
   - تأكد من البيانات المطلوبة
   - استخدم المعاينة أولاً

2. **عند التصدير**:
   - اختر الأعمدة المهمة فقط
   - استخدم Excel للبيانات الكاملة
   - استخدم PDF للتقارير الرسمية

3. **للتحليل**:
   - ابدأ بالتقرير الملخص
   - استعرض الإحصائيات
   - استخدم الرسوم البيانية للتوضيح
   - ارجع للتفاصيل عند الحاجة

4. **للعروض التقديمية**:
   - استخدم الرسوم البيانية
   - اطبع الملخصات الإحصائية
   - أضف جداول مختارة للتفاصيل

## 🆘 الدعم والمساعدة

### للحصول على المساعدة:
1. راجع هذا الدليل أولاً
2. تحقق من الأمثلة العملية
3. جرب الميزات المختلفة
4. تواصل مع فريق الدعم الفني

### المشاكل الشائعة وحلولها:

**مشكلة**: التقرير لا يُطبع
**الحل**: تأكد من تفعيل النوافذ المنبثقة في المتصفح

**مشكلة**: الأعمدة غير واضحة
**الحل**: قلل عدد الأعمدة المختارة أو استخدم Excel

**مشكلة**: الرسوم البيانية لا تظهر
**الحل**: تأكد من وجود بيانات كافية للتحليل
