## كيفية الاستخدام

### من واجهة المنصة:
1. اذهب إلى **الإعدادات** ⚙️
2. اختر **ألوان المنصة**
3. اضغط على أي مجموعة لونية
4. سيتم تطبيق الألوان فوراً وحفظها تلقائياً

### مميزات النظام:
- ✅ **حفظ تلقائي:** يتم حفظ اختيارك في المتصفح
- ✅ **تطبيق فوري:** الألوان تتغير مباشرة
- ✅ **معاينة مباشرة:** شاهد الألوان قبل الاختيار
- ✅ **دعم الوضع الداكن:** يتم ضبط الألوان تلقائياً

## البنية التقنية

### الملفات الأساسية:
- `src/config/themes.ts` - تعريف جميع المجموعات اللونية
- `src/context/ThemeContext.tsx` - إدارة حالة الثيم
- `src/app/pages/SettingsPage.tsx` - صفحة الإعدادات
- `src/styles/theme.css` - متغيرات CSS

### إضافة مجموعة جديدة:
```typescript
// في src/config/themes.ts
{
  id: 'theme-9',
  name: 'New Theme',
  nameAr: 'مجموعة جديدة',
  colors: {
    primary: '#HEXCODE',
    secondary: '#HEXCODE',
    accent: '#HEXCODE',
    background: '#HEXCODE'
  }
}
```

## متغيرات CSS المتاحة

يمكن استخدام هذه المتغيرات في أي ملف CSS:
```css
var(--primary)      /* اللون الأساسي */
var(--secondary)    /* اللون الثانوي */
var(--accent)       /* اللون المميز */
var(--background)   /* لون الخلفية */
var(--foreground)   /* لون النص (يتم حسابه تلقائياً) */
var(--destructive)  /* لون التحذيرات */
```

## ملاحظات
- النظام يحسب تلقائياً ما إذا كانت الخلفية داكنة أو فاتحة

- جميع الصفحات تستجيب فوراً لتغيير الألوان
