# ملاحظات التحويل إلى منصة عامة

## ما تم في هذه النسخة

- إزالة ملفات Electron والنسخة المحلية.
- تعديل package.json ليكون Frontend فقط.
- إضافة backend مستقل داخل مجلد `backend`.
- تجهيز Prisma schema مع PostgreSQL.
- تجهيز API أولي للصكوك والمرفقات.
- تحويل واجهة المرفقات إلى روابط Google Drive بدل رفع ملفات من الجهاز.
- إضافة ملف `.env.example` للواجهة و `.env.example` للـ Backend.

## ما لم يتم بعد

- ربط جميع الصفحات بالـ Backend؛ الحالي يربط الصكوك أولًا، وبقية الصفحات تبقى مؤقتًا على LocalStorage.
- إضافة تسجيل دخول حقيقي وصلاحيات على مستوى الخادم.
- إنشاء API لبقية الكيانات.
- ربط المرفقات فعليًا داخل كل نموذج حفظ. الواجهة الآن تعرض إدخال الروابط، والمرحلة التالية تحفظها عبر `/api/attachments`.

## التوصية التالية

ابدأ بتشغيل Backend محليًا، ثم نربط صفحة الصكوك بالكامل بالـ API، وبعدها نكرر نفس النمط لبقية الصفحات.
